    59→4.1 Lenis Smooth Scrolling Setup
    60→Initialize Lenis in a client-side provider wrapping the entire app
    61→Duration: 1.2s, easing: 0.075, 0.1, 0.1, 1 (smooth ease-out)
    62→Sync with GSAP ScrollTrigger: lenis.on('scroll', ScrollTrigger.update) and gsap.ticker.add((time) => lenis.raf(time * 1000))
    63→Disable on mobile when user prefers-reduced-motion is set
    64→Smooth scroll to anchors when navigation links are clicked
    65→Scroll to top on route change
    66→4.2 Text Animations (SplitType + GSAP)
    67→SplitTextReveal Component:
    68→
    69→Accepts props: text, type ('char' | 'word' | 'line'), animation ('slideUp' | 'fadeIn' | 'blurIn' | 'stagger'), stagger (number), duration (number), delay (number), trigger ('scroll' | 'mount')
    70→On scroll trigger: uses ScrollTrigger with start: 'top 85%', once: true
    71→On mount trigger: plays immediately with optional delay
    72→Default stagger: 0.03s for chars, 0.06s for words, 0.1s for lines
    73→Default duration: 0.8s
    74→Easing: power3.out
    75→TextBlurReveal Component:
    76→
    77→Text starts with opacity: 0, filter: blur(10px), transform: translateY(15px)
    78→Animates to opacity: 1, filter: blur(0px), transform: translateY(0)
    79→Duration: 0.9s, easing: power2.out
    80→Can be used on individual elements or as a stagger parent
    81→Scroll-triggered by default
    82→Headline Animation Standard:
    83→
    84→Every H1 on the site uses SplitTextReveal with type='line', animation='slideUp', stagger=0.12
